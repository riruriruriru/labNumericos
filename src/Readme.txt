mainE: contiene la ejecución y análisis de todos los métodos para sistemas de ecuaciones lineales
mainGestor.m: contiene la ejecución y análisis del uso del algoritmo gestor de decisión de métodos para ecuaciones lineales
mainSE.m: contiene la ejecución y análisis de el sistema de ecuaciones no lineal con el método Newton Raphson Multivariable

Todos los main cargan los datos que utilizan, ya sea la función de varias variables o las tres matrices de 289x289, 1089x1089 y 4225x4225.
En la carpeta Graficar se tienen las funciones necesarias para graficar los resultados obtenidos mediante la utilización de los métodos.
En la carpeta sistemas, se tienen los archivos .dat correspondientes a las 3 matrices que se utilizarán en este laboratorio.
En la carpeta Matrices, se tienen métodos para verificar propiedades y características de matrices, como por ejemplo si posee diagonal dominante o si es simétrica.
En la carpeta Métodos, se tienen los métodos utilizados para la resolución de sistemas de ecuaciones lineales y no lineales.
